
# Requirements

* node js

# Installation
 ```
npm install -g bower grunt-cli yo generator-angular
npm install
bower install
```
# Run
```
grunt serve  (dev mode)
grunt serve:dist (production mode)
```
# Test
```
grunt test
```

# Build and Deploy
```
deploy.sh
```
